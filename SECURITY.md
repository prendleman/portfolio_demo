# Security

This repository is a **public synthetic portfolio demo**. It is not a production service and must not be used to store secrets.

## Reporting

If you find a security problem in example scripts, CI, or documentation (for example, unsafe defaults or accidental secret handling), please open a **GitHub Issue** with details and a minimal reproduction. **Do not** paste live API keys, tokens, or proprietary data in issues or pull requests.

For coordinated disclosure that must stay private, use GitHub’s security reporting flow for this repository if it is enabled in the repo settings.
